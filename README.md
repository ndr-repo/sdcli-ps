# sdcli-ps - Subbdom CLI for PowerShell

## Quickstart
- Add the sdcli folder to environment variables/PATH
- Add your API key to sdcli/key.txt

## Syntax

### Print to stdout (for usage in scripts)
```
sdcli <target-domain> -
```

### Log results to file
```
sdcli <target-domain> <result-filename.extension>
```
